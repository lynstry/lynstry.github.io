# SoftEng-Research-Hub
 Студентський науковий гурток "SoftEng Research Hub" працює з метою створення умов для поглибленого вивчення студентами сучасних підходів до розробки програмного забезпечення, розвитку практичних навичок проєктної та командної роботи, підготовки студентів до участі в олімпіадах, конкурсах та реальних комерційних проєктах
